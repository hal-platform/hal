Hal - Version 2.4.2

What is Hal?

Hal is the deployment system we use to deploy many of our client-facing websites, including MyQL, Rocket Mortgage, QLMS, ORMS, Agent Insight, QuickenLoans.com and more!

Changelog

User Experience (UX) Enhancements

  • Add longer login sessions.

    Login session length has increased from 1 day to 5 days.

  • Updated faq to clarify application setup requirements.

Bugs

  • Deployment history link does not show all previous pushes were failures.

    Previously, the history icon would not show if all pushes for a deployment were failures.

    In addition, now the latest push will always be displayed on push status boxes, even failures.

  • Prevent doctrine cache generation from connecting to DB.

Technical

  • Created PHP WinRM library for connecting to windows servers.

Need Help?

If you find any bugs or want to request a feature, drop us a line in the hipchat room [Team] Web Core, or send us an email.
