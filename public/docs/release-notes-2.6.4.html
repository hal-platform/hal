HAL 9000 - Version 2.6.4

What is HAL 9000?

HAL 9000 is the deployment system we use to deploy many of our client-facing websites, including MyQL, Rocket Mortgage, QLMS, ORMS, Agent Insight, QuickenLoans.com and more!

Changelog

User Experience (UX) Enhancements

  • Email Notifications now use pretty names for deployments, which uses custom labels if set.
  • Add ability to filter kraken configuration history by environment.

Bugs

  • Enable kraken permission checks.
  • Fix error when deploying kraken as non-admin.
  • Allow leads to edit qks key settings for non prod environments.
  • When selecting an open PR when starting a build, the wrong tab is no longer selected.

Technical

  • Dependencies

    The following components MUST go with this release:

    • HAL 2.6.4
    • HAL Agent 2.4.4
    • HAL Core 2.8.1
