HAL 9000 - Version 2.6.3

What is HAL 9000?

HAL 9000 is the deployment system we use to deploy many of our client-facing websites, including MyQL, Rocket Mortgage, QLMS, ORMS, Agent Insight, QuickenLoans.com and more!

Changelog

User Experience (UX) Enhancements

  • Administrators can now view Encrypted Properties, unencrypted.

Technical

  • Kraken now uses mcp-crypto 2.1.2, which supports QKS Client Protocol and ensures interop between PHP and .NET.
  • Dependencies

    The following components MUST go with this release:

    • HAL 2.6.3
