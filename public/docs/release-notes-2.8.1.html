Hal - Version 2.8.1

What is Hal?

Hal is the deployment system we use to deploy many of our client-facing websites, including MyQL, Rocket Mortgage, QLMS, ORMS, Agent Insight, QuickenLoans.com and more!

Changelog

Enhancement

  • Add support for dash accounts
    Dash accounts are missing many details that normal accounts have including Full name, Email, and ID. Previously, Hal required these parameters. New accounts will now use a random ID rather than common ID from AD.
